## ワーク1: 会話の3つの型（3分）

以下の**3つの型**を覚えてください。

### 型1: 質問型（相手に興味を持つ）

相手のことを知るための質問をする。

**例:**
```

```

### 型2: 共感型（相手の話に共感する）

相手の話に共感することで、親近感を与える。

**例:**
```
わかります！僕もそう思います！
```

### 型3: 自己開示型（自分の話もする）

自分の話もすることで、信頼関係を築く。

**例:**
```
実は僕も、○○が好きなんです。
```
