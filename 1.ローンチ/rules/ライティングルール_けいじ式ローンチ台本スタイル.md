# ライティングルール：けいじ式ローンチ台本スタイル

**最終更新日：2025-11-08**

本ドキュメントは、「けいじ」のローンチ台本（1話～7話、プロフィールストーリー、ミッション・ビジョン、個別相談誘導動画）を分析し、同じような文章を再現するための書き方や文体のルールを体系化したものです。

---

## 1. 全体構成・フォーマット

### 1.1 タイトル構造
- **【】括弧**を使用してセクション名を明確化（例：【興味付け】【問題の原因提示】）
- タイトルには**数字・問いかけ・衝撃性**を含める（例：「たった1つの方法」「衝撃の理由」「3つの共通点」）
- サブタイトルでは「〜とは？」「なぜ〜なのか？」といった**疑問形**を多用

### 1.2 セクション構成パターン
ローンチ動画の典型的な流れ：
1. 【興味付け】- 冒頭で強烈なフックをかける
2. 【問題提示・共感】- 読者の悩みを言語化して共感を得る
3. 【原因の分析】- 問題の本質的な原因を明らかにする
4. 【新しい視点の提示】- 従来の常識を覆す新概念を提示
5. 【解決策の提示】- 具体的なメソッド・ロードマップを示す
6. 【証拠・実績】- 成功事例や根拠を示す
7. 【まとめ・次回予告】- 振り返りと次への期待感醸成
8. 【CTA（行動喚起）】- 勉強会や個別相談への誘導

### 1.3 マークダウン使用ルール
- **太字（\*\*）**：重要な主張、キーワード、読者に強調したい部分
- ★マーク：特に重要な情報、具体例、データポイント
- 箇条書き（・）：並列情報、具体例の列挙
- 改行：文章のリズムを作るため、1～2文ごとに改行を入れる

## 2. トーン＆ボイス（文体・語り口）

### 2.1 基本の語り口
- **一人称**：「僕」で統一（「私」は使わない）
- **二人称**：「あなた」で統一し、常に直接語りかける
- **敬語レベル**：「です・ます調」だが、くだけた口語表現も混在させる
  - 例：「〜じゃないですか」「〜ですよね？」「〜なんですよ」「〜なんです」

### 2.2 会話的な表現技法
- **問いかけ形式を多用**する
  - 「〜って思いませんか？」
  - 「でも、不思議に思いませんか？」
  - 「じゃあどうすればいいのか？」
  - 「その答えは何でしょうか？」
- **自問自答パターン**を使う
  - 読者の疑問を先読みして代弁：「『けいじさん、でも〜じゃないですか？』って思う方もいると思いますが…」
  - 即座に回答を返す：「安心してください。〜です。」
- **共感フレーズ**を随所に入れる
  - 「僕もそうでした」
  - 「この気持ちはよくわかります」
  - 「無理もありません」

### 2.3 リズム・テンポの作り方
- **短文と長文を交互に配置**してリズムを作る
  - 短文：結論や強調（1～2行）
  - 長文：説明や具体例（3～5行）
- **体言止め**を効果的に使う（例：「それが、唯一無二の男になるということ。」）
- **「…」（三点リーダー）**で間を作り、次の展開への期待感を演出
- **倒置法**を使って強調する（例：「それは、勇気です。」）

## 3. 説得・エンゲージメント技法

### 3.1 問題提起と共感のパターン
**ステップ1：読者の悩みを具体的に列挙**
```
・好きな女性がいたけど勇気を出してアプローチができなかったり
・好きな女性と連絡はできていたけど、連絡が返ってこなくなってしまったり
・好きな女性とデートはできたけど、「友達としてはいい人なんだけど」と恋愛対象になれなかったり
```
- 箇条書きで**3～7個**のパターンを示す
- 「〜したり」で文末を揃えてリズムを作る
- 読者が「自分のことだ」と感じる具体性を持たせる

**ステップ2：共感を示す**
```
僕も同じことを思ってました。
めちゃくちゃ悔しくて、自分を責めて、ダメな自分を変えたくて、いろんな恋愛本や情報商材を買い漁りました。
```
- 「僕も〜」で自分も同じ経験をしたことを示す
- 感情を具体的に描写する（悔しい、嫌気がさす、陥る）

### 3.2 逆説・常識の否定パターン
従来の常識を否定し、新しい視点を提示する際の型：
```
「〜って思った方がいると思いますが、実は違うんです。」
「でも、もし〜だとしたら…？」
「そう、実はあなたが〜していることこそが、〜の原因になっているんです。」
```
- **「でも」「しかし」「実は」**で逆接を明示
- **「〜だとしたら？」**で仮定を投げかけて考えさせる
- **「そう。」**という一言で区切り、重要な真実を告げる

### 3.3 例え話・メタファーの使い方
抽象的な概念を具体的なイメージに変換する：

**基本構造**
```
例えば、〜って、〜みたいなものですよね。
これは、〜に例えるとわかりやすいです。
〜で言えば、〜みたいな感じです。
```

**具体的なメタファー例**
- 沖縄旅行の例：戦略の重要性を説明
- ライザップの例：ノウハウと実践の違いを説明
- サッカー・野球の例：失敗と改善の必要性を説明
- 穴の開いたバケツ：努力が無駄になる状態を説明
- ダイヤモンド・フェラーリ：希少性の価値を説明

**メタファーの選び方**
- 誰もが知っている有名な事例・商品・スポーツを選ぶ
- 読者の生活や経験に近いものを選ぶ
- ビジネス・スポーツ・日常生活から幅広く引用

### 3.4 権威性・証拠の示し方
**数字で具体性を出す**
- 「1万人以上の女性にアプローチ」
- 「300人以上の恋愛相談」
- 「2190時間の恋愛活動時間」
- 「1000万以上の自己投資」
- 「20代男性の4割が一度も彼女ができたことがない」

**書籍・理論の引用**
- 「アドラーの『嫌われる勇気』でも…」
- 「リチャード・ドーキンス博士の『利己的な遺伝子』にもあるように…」
- 「幸福の資本論」
- 学問名を列挙して専門性を示す（脳科学・認知科学・深層心理学・進化心理学・行動心理学など）

**成功事例を示す**
- 年齢・状況を具体的に示す（例：「40代後半の会社員、彼女いない歴年齢、童貞」）
- ビフォーアフターを明確にする
- 複数の事例をテンポよく列挙する

## 4. 感情を動かす文章技法

### 4.1 痛みを掘り下げる（Problem Agitation）
読者の痛みを深掘りして、現状を変えたい気持ちを強める：

**パターン1：最悪の未来を想像させる**
```
1年後も今と何も状況が変わらないどころか、どんどん自信をなくしていく…
そんな未来、嫌ですよね？
```

**パターン2：機会損失を示す**
```
あなたが「自分には無理だ」と恋愛を諦めてしまうことで、
本来、あなたのその誠実さで幸せになるはずだった一人の女性が、
自己中心的で無責任な男の手に渡り、心も体も傷つけられてしまうかもしれない。
```

**パターン3：時間の制約を示す**
```
この期限を過ぎてしまうと、二度とこの動画講座を見ることはできません。
```

### 4.2 希望を提示する（Solution）
痛みの後に必ず希望を示す：

**パターン1：安心させる**
```
安心してください。
〜という方法があります。
```

**パターン2：可能性を示す**
```
でも、〜ができれば、あなたは〜という未来を手に入れることができます。
```

**パターン3：すでに持っているものに気づかせる**
```
実は、あなたの中にすべての要素はすでにあるんです。
だからあなたを変える必要はない。
```

### 4.3 理想の未来を描写する
五感を刺激する具体的な描写で、理想の未来をリアルに想像させる：

```
・一緒に同棲生活をして、仕事から帰ってきたら好きな女性が笑顔で迎えてくれる毎日
・週末、行きつけのカフェやデートでまったり過ごしたり
・お互いの家でゆっくりネットフリックスで映画を見たり
・ドライブで絶景の夜景を一緒に見たり
```
- 箇条書きで5～10個のシーンを列挙
- 「〜したり」で統一してリズムを作る
- 日常的なシーンから非日常的なシーンまで幅広く描く

## 5. ストーリーテリング（物語性）

### 5.1 自己開示のパターン
**過去の失敗・痛みを語る**
```
僕も元々〜でした。
〜という状態で、めちゃくちゃ悔しい思いをしてきました。
でも、〜に気づいたとき、僕の人生は劇的に変わっていきました。
```
- ビフォー（過去の失敗）→ 転機（気づき）→ アフター（変化）の3段階で語る
- 感情を具体的に描写する
- 「あなたと同じだった」ことを強調して共感を得る

### 5.2 成功事例の語り方
**具体性とリアリティを重視**
```
例えば、40代後半の会社員、Mさん。
彼は40代にも関わらず、彼女いない歴年齢で、童貞の状態でした。
彼は、恋愛になると極度にコンフォートゾーンが狭く、
外見磨きやマッチングアプリを始めることさえままならない状態でした。

（中略：取り組みの内容）

今では彼は、マッチングアプリを使いこなし、
毎月何人もの女性とデートをし、
最初からは信じられないレベルまでコミュニケーション力と自信がアップしたんですよね。
```
- 年齢・職業・状況を具体的に示す
- 問題の深刻さを強調する
- 取り組み内容を簡潔に示す
- ビフォーアフターを対比させる
- 本人の声（感想）を入れる

### 5.3 共通の敵を設定する
読者と同じ立場に立ち、外部の敵を設定して一体感を作る：

**敵の例**
- 再現性のないノウハウを売る恋愛講師
- 陽キャ仕様のテクニックを押し付ける業界
- 誠実な男性を傷つける自己中心的な男たち
- 女性を大切にしない風潮

**敵の設定方法**
```
悪いのはあなたじゃありません。
悪いのは〜です。

なぜなら、彼らは〜だからです。
```
- 「あなたは悪くない」と免罪する
- 外部に責任を転嫁する
- その理由を論理的に説明する
- 「だからこそ、僕は〜」と自分のミッションにつなげる

## 6. 論理構成・説明パターン

### 6.1 結論ファーストの原則
重要なポイントは必ず結論から述べる：
```
早速、結論から言います。
あなたの恋愛がうまくいかない本当の理由。
それは、〜です。
```

### 6.2 三段論法・ロジックツリー
複雑な概念を分解して説明する：
```
恋愛の成功 = アプローチ数 × 成功率

成功率は、以下の4つに細分化できます：
- マッチ率
- 初デート移行率
- 2回目デート移行率
- クロージング率
```
- 数式・図式化できるものは構造を明示する
- 要素を分解して説明する
- 各要素を箇条書きで示す

### 6.3 ステップ・フレームワークの提示
行動を明確なステップに分解する：
```
けいじ式ロードマップは、以下の3ステップです：

1. ロジカルに戦略を立てる
2. コンフォートゾーンを超えた行動をする
3. フィードバックをもらって即改善する
```
- ステップは3～7個にまとめる（3が最も多い）
- 各ステップを動詞で始める
- 各ステップを後で詳しく説明する

### 6.4 理由の説明パターン
「なぜ〜なのか？」に対する答え方：
```
なぜ、〜なのか？
それは、〜だからです。

（具体例・証拠）

つまり、〜ということです。
```
- 問いかけを先に提示
- 回答を簡潔に示す
- 具体例や証拠で補強
- 「つまり」で要約する

## 7. CTA（行動喚起）のパターン

### 7.1 希少性・緊急性の演出
```
ただし、この〜は、期間限定の案内となっています。
この期限を過ぎてしまうと、二度と〜することはできません。
```
- 期間限定を明示
- 「二度と〜できない」で損失回避心理を刺激

### 7.2 ハードル下げ＋価値強調
```
料金は、1万円ですが、
この動画を見てくださっている方は、期間限定で無料にします。
```
- 本来の価値（価格）を先に示す
- 特別扱いであることを強調
- 無料・割引を提示

### 7.3 行動を具体的に指示
```
興味のある方は今すぐ視聴ページの下のボタンからご参加ください！
```
- 「今すぐ」で即時性を促す
- 「〜から」と具体的な方法を示す
- 「！」で強調する

### 7.4 参加資格の明示（ネガティブスクリーニング）
```
ただし、
・情報だけもらって行動する気がない方
・女性とヤれればいいと考える自己中な方
・ただ話を聞いてほしいだけの方
こういった方は、この会に参加しても、結果が出る可能性が低く、
お互い時間の無駄になってしまうので、ご遠慮いただきたいです。
```
- 「ただし」で条件を示す
- 来てほしくない人を明示（逆説的に真剣な人を引き寄せる）
- 理由を添える

## 8. 文末表現・語尾のバリエーション

### 8.1 断定表現
- 「〜です。」
- 「〜なんです。」
- 「〜ということです。」
- 「〜だからです。」

### 8.2 問いかけ表現
- 「〜ですよね？」
- 「〜じゃないですか。」
- 「〜だと思いませんか？」
- 「〜でしょうか？」

### 8.3 共感・確認表現
- 「〜ですよね。」
- 「〜だと思います。」
- 「〜かもしれません。」
- 「〜のではないでしょうか。」

### 8.4 強調表現
- 「〜なんですよ。」
- 「〜なんですよね。」
- 「〜わけです。」
- 「〜ということなんです。」

### 8.5 口語的表現
- 「〜ってことです。」
- 「〜みたいな感じです。」
- 「〜って思いますよね。」
- 「〜じゃん」（カジュアルな場面のみ）

## 9. 禁止事項・避けるべき表現

### 9.1 避けるべき表現
- ❌ 過度に堅苦しい表現（「〜である」「〜する必要がある」など論文調）
- ❌ 上から目線の命令口調（「〜しなさい」「〜すべきだ」）
- ❌ 抽象的すぎる表現（具体例がない概念の羅列）
- ❌ 専門用語の乱用（説明なしに難しい用語を使う）
- ❌ ネガティブな断定（「あなたには無理」「諦めるべき」など）

### 9.2 注意すべきポイント
- ⚠️ 「〜と思います」を連発しない（自信がなさそうに見える）
- ⚠️ 同じ文末表現を3回以上連続させない（単調になる）
- ⚠️ 一文が3行以上になる場合は分割を検討
- ⚠️ 箇条書きが10個以上になる場合はグルーピングを検討

## 12. まとめ

このライティングルールの核心は、以下の3つです：

1. **会話的で親しみやすい語り口**：「あなた」に直接語りかけ、問いかけと共感で引き込む
2. **論理と感情のバランス**：ステップ・数字・理論で論理性を担保しつつ、ストーリーと理想の未来で感情を動かす
3. **具体性とリアリティ**：抽象論で終わらせず、常に具体例・メタファー・数字で補強する

これらの要素を組み合わせることで、「けいじ式」の説得力があり、読者を行動に駆り立てる文章を再現できます。

**重要な心構え**：
- 書き手（けいじ）は読者の味方であり、共に戦う同志である
- 読者の痛みに寄り添い、希望を示し、具体的な道筋を示す
- 常に「あなた」の成功と幸せを願っている姿勢を貫く

このルールを守りつつ、扱うテーマに応じて柔軟に内容を変えることで、
様々なトピックに対応した「けいじ式」の文章を作成できます。
