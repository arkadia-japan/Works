# ライティングルール：けいじ式ローンチ台本スタイル

**最終更新日：2025-11-15**

本ドキュメントは、「けいじ」のローンチ台本（1話～7話、プロフィールストーリー、ミッション・ビジョン、個別相談誘導動画）を分析し、同じような文章を再現するための書き方や文体のルールを体系化したものです。

---

## 1. 全体構成・フォーマット

### 1.1 タイトル構造
- **【】括弧**を使用してセクション名を明確化（例：【興味付け】【問題の原因提示】）
- タイトルには**数字・問いかけ・衝撃性**を含める（例：「たった1つの方法」「衝撃の理由」「3つの共通点」）
- サブタイトルでは「〜とは？」「なぜ〜なのか？」といった**疑問形**を多用

### 1.2 セクション構成パターン
ローンチ動画の典型的な流れ：
1. 【興味付け】- 冒頭で強烈なフックをかける
2. 【問題提示・共感】- 読者の悩みを言語化して共感を得る
3. 【原因の分析】- 問題の本質的な原因を明らかにする
4. 【新しい視点の提示】- 従来の常識を覆す新概念を提示
5. 【解決策の提示】- 具体的なメソッド・ロードマップを示す
6. 【証拠・実績】- 成功事例や根拠を示す
7. 【まとめ・次回予告】- 振り返りと次への期待感醸成
8. 【CTA（行動喚起）】- 勉強会や個別相談への誘導

### 1.3 マークダウン使用ルール
- **太字（\*\*）**：重要な主張、キーワード、読者に強調したい部分
- ★マーク：特に重要な情報、具体例、データポイント
- 箇条書き（・）：並列情報、具体例の列挙
- 改行：文章のリズムを作るため、1～2文ごとに改行を入れる

## 2. トーン＆ボイス（文体・語り口）

### 2.1 基本の語り口
- **一人称**：「僕」で統一（「私」は使わない）
- **二人称**：「あなた」で統一し、常に直接語りかける
- **敬語レベル**：「です・ます調」だが、くだけた口語表現も混在させる
  - 例：「〜じゃないですか」「〜ですよね？」「〜なんですよ」「〜なんです」

### 2.2 会話的な表現技法
- **問いかけ形式を多用**する
  - 「〜って思いませんか？」
  - 「でも、不思議に思いませんか？」
  - 「じゃあどうすればいいのか？」
  - 「その答えは何でしょうか？」
  - **★重要ポイント前の問いかけパターン**：
    - 「何だと思いますか？」→（改行）→「それは、〜です。」
    - 「これはなぜか？」→（改行）→「〜だからです。」
    - 「どれも1つだけ共通していることがあるんです。何だと思いますか？」
  - **問いかけ後の間の取り方**：重要な答えを提示する前に必ず改行を入れて、読者に考える時間を与える
- **自問自答パターン**を使う
  - 読者の疑問を先読みして代弁：「『けいじさん、でも〜じゃないですか？』って思う方もいると思いますが…」
  - 即座に回答を返す：「安心してください。〜です。」
  - 読者の心の声を代弁する：「それがわかれば苦労しねぇよって思ってる方もいると思います」「なかなか自分の戦略って考えれないですよね？」
- **共感フレーズ**を随所に入れる
  - 「僕もそうでした」
  - 「この気持ちはよくわかります」
  - 「無理もありません」
  - 「僕も非モテ時代はまさしく同じ感じでした」
  - 「そんな方に背中を押す意味で」

### 2.3 リズム・テンポの作り方
- **短文と長文を交互に配置**してリズムを作る
  - 短文：結論や強調（1～2行）
  - 長文：説明や具体例（3～5行）
- **体言止め**を効果的に使う（例：「それが、唯一無二の男になるということ。」）
- **「…」（三点リーダー）**で間を作り、次の展開への期待感を演出
- **倒置法**を使って強調する（例：「それは、勇気です。」）
- **「そう」「つまり」などの区切りワード**を使って文章にリズムと強調を作る
  - **「そう。」**：重要な真実を告げる前に使う（例：「そう。男って恋愛がうまくいってないと〜」）
  - **「つまり、」**：論点を整理し要約する時に使う（例：「つまり、やる気もあるし焦りもあるけれども〜」）
  - **「これはなぜか？」**：理由を説明する前に使う
  - **「やばいですよね？」「これ、異常だと思いませんか？」**：問題の深刻さを強調
  - 区切りワードの後は改行を入れて、次の内容を際立たせる

## 3. 説得・エンゲージメント技法

### 3.1 問題提起と共感のパターン
**ステップ1：読者の悩みを具体的に列挙**
```
・好きな女性がいたけど勇気を出してアプローチができなかったり
・好きな女性と連絡はできていたけど、連絡が返ってこなくなってしまったり
・好きな女性とデートはできたけど、「友達としてはいい人なんだけど」と恋愛対象になれなかったり
```
- 箇条書きで**3～7個**のパターンを示す
- 「〜したり」で文末を揃えてリズムを作る
- 読者が「自分のことだ」と感じる具体性を持たせる
- **★具体的な相談事例を引用する**：抽象的な質問ではなく、実際に届いた相談メッセージを長文で引用することで、リアリティと臨場感を出す
  - 例：「僕の動画講座を見てくれた方がLINEにてこんな相談をしてくれました。『〜（長文の相談内容）〜』」
  - 読者の内面の葛藤（「電話して嫌われたくない」「話すことがなくなったらどうしよう」）まで描写すると共感度が上がる

**ステップ2：共感を示す**
```
僕も同じことを思ってました。
めちゃくちゃ悔しくて、自分を責めて、ダメな自分を変えたくて、いろんな恋愛本や情報商材を買い漁りました。
```
- 「僕も〜」で自分も同じ経験をしたことを示す
- 感情を具体的に描写する（悔しい、嫌気がさす、陥る）

### 3.2 逆説・常識の否定パターン
従来の常識を否定し、新しい視点を提示する際の型：
```
「〜って思った方がいると思いますが、実は違うんです。」
「でも、もし〜だとしたら…？」
「そう、実はあなたが〜していることこそが、〜の原因になっているんです。」
```
- **「でも」「しかし」「実は」**で逆接を明示
- **「〜だとしたら？」**で仮定を投げかけて考えさせる
- **「そう。」**という一言で区切り、重要な真実を告げる

### 3.3 例え話・メタファーの使い方
抽象的な概念を具体的なイメージに変換する：

**基本構造**
```
例えば、〜って、〜みたいなものですよね。
これは、〜に例えるとわかりやすいです。
〜で言えば、〜みたいな感じです。
```

**具体的なメタファー例**
- 沖縄旅行の例：戦略の重要性を説明
- ライザップの例：ノウハウと実践の違いを説明
- サッカー・野球の例：失敗と改善の必要性を説明
- 穴の開いたバケツ：努力が無駄になる状態を説明
- ダイヤモンド・フェラーリ：希少性の価値を説明
- **★家の内見と結婚の対比**：「家は何度も内見したり誰かに相談して決めるのに、結婚はノリと勢いでしてしまう人がなんと多いことか」
- **★地図が間違っている状態**：戦略ミスの深刻さを説明「旅行に行くときの地図が間違っている状態なので、いつまでたっても目的地にたどり着けない」
- **★お金持ちがキャバクラに行く理由**：恋愛コンプレックスの深刻さを説明「多くのお金持ちはキャバクラやホステスに好んでいく。彼らはそこで毎日数十万円単位のお金を使います。これはなぜか？恋愛に対してコンプレックスがあるからです」

**メタファーの選び方**
- 誰もが知っている有名な事例・商品・スポーツを選ぶ
- 読者の生活や経験に近いものを選ぶ
- ビジネス・スポーツ・日常生活から幅広く引用
- **対比を使う**：2つの行動（家の内見 vs 結婚）を対比させることで、矛盾を際立たせる

### 3.4 権威性・証拠の示し方
**数字で具体性を出す**
- 「1万人以上の女性にアプローチ」
- 「300人以上の恋愛相談」
- 「2190時間の恋愛活動時間」
- 「1000万以上の自己投資」
- 「20代男性の4割が一度も彼女ができたことがない」

**書籍・理論の引用**
- 「アドラーの『嫌われる勇気』でも…」
- 「リチャード・ドーキンス博士の『利己的な遺伝子』にもあるように…」
- 「幸福の資本論」
- 学問名を列挙して専門性を示す（脳科学・認知科学・深層心理学・進化心理学・行動心理学など）

**成功事例を示す**
- 年齢・状況を具体的に示す（例：「40代後半の会社員、彼女いない歴年齢、童貞」）
- ビフォーアフターを明確にする
- 複数の事例をテンポよく列挙する

## 4. 感情を動かす文章技法

### 4.1 痛みを掘り下げる（Problem Agitation）
読者の痛みを深掘りして、現状を変えたい気持ちを強める：

**パターン1：最悪の未来を想像させる**
```
1年後も今と何も状況が変わらないどころか、どんどん自信をなくしていく…
そんな未来、嫌ですよね？
```

**パターン2：機会損失を示す**
```
あなたが「自分には無理だ」と恋愛を諦めてしまうことで、
本来、あなたのその誠実さで幸せになるはずだった一人の女性が、
自己中心的で無責任な男の手に渡り、心も体も傷つけられてしまうかもしれない。
```

**パターン3：時間の制約を示す**
```
この期限を過ぎてしまうと、二度とこの動画講座を見ることはできません。
```

**パターン4：具体的な痛みを列挙する**
```
でも、結局うまくいかず離婚したり、ATMとして使われるだけの男になったり、めちゃくちゃ苦労している人が多いんですよね。
```
- 「〜したり」で統一してリズムを作る
- 具体的な最悪のシナリオを3〜5個列挙する
- 「ATMとして使われる」など、読者が恐れている具体的な状況を描写

### 4.2 希望を提示する（Solution）
痛みの後に必ず希望を示す：

**パターン1：安心させる**
```
安心してください。
〜という方法があります。
```

**パターン2：可能性を示す**
```
でも、〜ができれば、あなたは〜という未来を手に入れることができます。
```

**パターン3：すでに持っているものに気づかせる**
```
実は、あなたの中にすべての要素はすでにあるんです。
だからあなたを変える必要はない。
```

### 4.3 理想の未来を描写する
五感を刺激する具体的な描写で、理想の未来をリアルに想像させる：

```
・一緒に同棲生活をして、仕事から帰ってきたら好きな女性が笑顔で迎えてくれる毎日
・週末、行きつけのカフェやデートでまったり過ごしたり
・お互いの家でゆっくりネットフリックスで映画を見たり
・ドライブで絶景の夜景を一緒に見たり
```
- 箇条書きで5～10個のシーンを列挙
- 「〜したり」で統一してリズムを作る
- 日常的なシーンから非日常的なシーンまで幅広く描く

## 5. ストーリーテリング（物語性）

### 5.1 自己開示のパターン
**過去の失敗・痛みを語る**
```
僕も元々〜でした。
〜という状態で、めちゃくちゃ悔しい思いをしてきました。
でも、〜に気づいたとき、僕の人生は劇的に変わっていきました。
```
- ビフォー（過去の失敗）→ 転機（気づき）→ アフター（変化）の3段階で語る
- 感情を具体的に描写する
- 「あなたと同じだった」ことを強調して共感を得る

**自分の体験談を具体的に語る**
```
逆に、理想のパートナーと出会えれば、毎日が幸せで満たされ、仕事も人生も前向きになります。
僕も今の妻と出会ってから、収入が一気に上がりました。
```
- 抽象的な説明だけでなく、自分の具体的な体験を追加する
- 数値や結果（「収入が上がった」など）を示すことで説得力が増す

### 5.2 成功事例の語り方
**具体性とリアリティを重視**
```
例えば、40代後半の会社員、Mさん。
彼は40代にも関わらず、彼女いない歴年齢で、童貞の状態でした。
彼は、恋愛になると極度にコンフォートゾーンが狭く、
外見磨きやマッチングアプリを始めることさえままならない状態でした。

（中略：取り組みの内容）

今では彼は、マッチングアプリを使いこなし、
毎月何人もの女性とデートをし、
最初からは信じられないレベルまでコミュニケーション力と自信がアップしたんですよね。
```
- 年齢・職業・状況を具体的に示す
- 問題の深刻さを強調する
- 取り組み内容を簡潔に示す
- ビフォーアフターを対比させる
- 本人の声（感想）を入れる

### 5.3 共通の敵を設定する
読者と同じ立場に立ち、外部の敵を設定して一体感を作る：

**敵の例**
- 再現性のないノウハウを売る恋愛講師
- 陽キャ仕様のテクニックを押し付ける業界
- 誠実な男性を傷つける自己中心的な男たち
- 女性を大切にしない風潮

**敵の設定方法**
```
悪いのはあなたじゃありません。
悪いのは〜です。

なぜなら、彼らは〜だからです。
```
- 「あなたは悪くない」と免罪する
- 外部に責任を転嫁する
- その理由を論理的に説明する
- 「だからこそ、僕は〜」と自分のミッションにつなげる

## 6. 論理構成・説明パターン

### 6.1 結論ファーストの原則
重要なポイントは必ず結論から述べる：
```
早速、結論から言います。
あなたの恋愛がうまくいかない本当の理由。
それは、〜です。
```

### 6.2 三段論法・ロジックツリー
複雑な概念を分解して説明する：
```
恋愛の成功 = アプローチ数 × 成功率

成功率は、以下の4つに細分化できます：
- マッチ率
- 初デート移行率
- 2回目デート移行率
- クロージング率
```
- 数式・図式化できるものは構造を明示する
- 要素を分解して説明する
- 各要素を箇条書きで示す

### 6.3 ステップ・フレームワークの提示
行動を明確なステップに分解する：
```
けいじ式ロードマップは、以下の3ステップです：

1. ロジカルに戦略を立てる
2. コンフォートゾーンを超えた行動をする
3. フィードバックをもらって即改善する
```
- ステップは3～7個にまとめる（3が最も多い）
- 各ステップを動詞で始める
- 各ステップを後で詳しく説明する

### 6.4 理由の説明パターン
「なぜ〜なのか？」に対する答え方：
```
なぜ、〜なのか？
それは、〜だからです。

（具体例・証拠）

つまり、〜ということです。
```
- 問いかけを先に提示
- 回答を簡潔に示す
- 具体例や証拠で補強
- 「つまり」で要約する

## 7. CTA（行動喚起）のパターン

### 7.1 希少性・緊急性の演出
```
ただし、この〜は、期間限定の案内となっています。
この期限を過ぎてしまうと、二度と〜することはできません。
```
- 期間限定を明示
- 「二度と〜できない」で損失回避心理を刺激

### 7.2 ハードル下げ＋価値強調
```
料金は、1万円ですが、
この動画を見てくださっている方は、期間限定で無料にします。
```
- 本来の価値（価格）を先に示す
- 特別扱いであることを強調
- 無料・割引を提示

### 7.3 行動を具体的に指示
```
興味のある方は今すぐ視聴ページの下のボタンからご参加ください！
```
- 「今すぐ」で即時性を促す
- 「〜から」と具体的な方法を示す
- 「！」で強調する

### 7.4 参加資格の明示（ネガティブスクリーニング）
```
ただし、
・情報だけもらって行動する気がない方
・女性とヤれればいいと考える自己中な方
・ただ話を聞いてほしいだけの方
こういった方は、この会に参加しても、結果が出る可能性が低く、
お互い時間の無駄になってしまうので、ご遠慮いただきたいです。
```
- 「ただし」で条件を示す
- 来てほしくない人を明示（逆説的に真剣な人を引き寄せる）
- 理由を添える

### 7.5 オファー前の不安の解消
```
でも、なかなか自分の戦略って考えれないですよね？
それがわかれば苦労しねぇよって思ってる方もいると思います。

安心してください。

今回、特別なご提案を用意しました。
```
- オファーを提示する前に、読者の心の声（「自分で考えるのは難しい」）を代弁
- 「安心してください」で不安を解消してからオファーを提示
- 読者が「YES」と言いやすい心理状態を作る

## 9. 禁止事項・避けるべき表現

### 9.1 避けるべき表現
- ❌ 過度に堅苦しい表現（「〜である」「〜する必要がある」など論文調）
- ❌ 上から目線の命令口調（「〜しなさい」「〜すべきだ」）
- ❌ 抽象的すぎる表現（具体例がない概念の羅列）
- ❌ 専門用語の乱用（説明なしに難しい用語を使う）
- ❌ ネガティブな断定（「あなたには無理」「諦めるべき」など）
- ❌ 企業寄りの抽象的なミッション・ビジョン説明の長文化（読者が知りたいのは「自分にどう関係があるか」）

### 9.2 注意すべきポイント
- ⚠️ 「〜と思います」を連発しない（自信がなさそうに見える）
- ⚠️ 同じ文末表現を3回以上連続させない（単調になる）
- ⚠️ 一文が3行以上になる場合は分割を検討
- ⚠️ 箇条書きが10個以上になる場合はグルーピングを検討
- ⚠️ 箇条書きの語尾は統一する（「〜したり」「〜される」など）

### 9.3 削減すべき冗長な説明
- ビジョン・バリュー・対象領域などの詳細すぎる企業説明
- 読者にとって重要度の低い情報（企業の全体像よりも「あなたにとってのメリット」を優先）
- 同じ内容の繰り返し（特に追伸セクションでのミッション再説明）
- **焦点を絞る**：企業のミッション→視聴者へのメッセージに変換する

## 12. まとめ

このライティングルールの核心は、以下の3つです：

1. **会話的で親しみやすい語り口**：「あなた」に直接語りかけ、問いかけと共感で引き込む
2. **論理と感情のバランス**：ステップ・数字・理論で論理性を担保しつつ、ストーリーと理想の未来で感情を動かす
3. **具体性とリアリティ**：抽象論で終わらせず、常に具体例・メタファー・数字で補強する

これらの要素を組み合わせることで、「けいじ式」の説得力があり、読者を行動に駆り立てる文章を再現できます。

**重要な心構え**：
- 書き手（けいじ）は読者の味方であり、共に戦う同志である
- 読者の痛みに寄り添い、希望を示し、具体的な道筋を示す
- 常に「あなた」の成功と幸せを願っている姿勢を貫く

このルールを守りつつ、扱うテーマに応じて柔軟に内容を変えることで、
様々なトピックに対応した「けいじ式」の文章を作成できます。

## 13. ブラッシュアップで追加されたルール（2025-11-15更新）

### 13.1 問いかけ強化テクニック
**重要ポイント前の問いかけパターン**
```
それぞれレベル感は違いますが、どれも1つだけ共通していることがあるんです。
何だと思いますか？

それは、**勇気**です。
```
- 答えを直接提示するのではなく、まず問いかけて読者に考えさせる
- 問いかけと答えの間に必ず改行を入れる
- 読者の注意を引き、次の答えへの期待感を高める

### 13.2 区切りワードの戦略的活用
**「そう」「つまり」「これはなぜか？」の使い方**
- **「そう。」**：重要な真実を告げる前（例：「そう。男って恋愛がうまくいってないと一生涯影響を及ぼすんですよね」）
- **「つまり、」**：論点を整理し要約（例：「つまり、やる気もあるし焦りもあるけれども、うまく行動に移せない」）
- **「これはなぜか？」**：理由説明の前（例：「これはなぜか？恋愛に対してコンプレックスがあるからです」）
- 区切りワードの後は改行を入れて、次の内容を際立たせる
- 文章全体で15箇所程度使用すると、リズムが生まれる

### 13.3 具体的な相談事例の活用
**抽象的な質問→長文の相談引用**
```
僕の動画講座を見てくれた方がLINEにてこんな相談をしてくれました。

「今回動画講座を見て、再現性を追求したノウハウにとても感銘を受けました。
これなら恋愛経験が少なくビビりな自分でも強みを活かして取り組めると思いました。
（中略：具体的な悩みの内容）
焦りはあるのですが本気の行動に移せてない自分がいます。」
```
- 一般論ではなく、実際の相談メッセージを長文で引用
- 読者の内面の葛藤まで描写する
- リアリティと臨場感が増し、「自分と同じだ」と感じやすくなる

### 13.4 対比を使った例え話
**2つの行動を対比させて矛盾を際立たせる**
```
家は何度も内見したり誰かに相談して決めるのに、結婚はノリと勢いでしてしまう人がなんと多いことか。
```
- 慎重な行動（家の内見）vs 軽率な行動（結婚）を対比
- 読者が日常的に経験する行動と比較することで、問題の深刻さが際立つ
- 「なんと多いことか」で感情を込める

### 13.5 自己否定の複数パターン免罪
**より多くの読者を安心させる**
```
それはあなたに「やる気がない」からでも「自信がない」からでもないんです。
```
- 1つではなく複数の自己否定パターンを否定する
- より多くの読者が「自分は悪くないんだ」と感じられる
- 「でも」を繰り返して強調

### 13.6 読者の心の声を代弁する
**オファー前の不安解消**
```
でも、なかなか自分の戦略って考えれないですよね？
それがわかれば苦労しねぇよって思ってる方もいると思います。

安心してください。

今回、特別なご提案を用意しました。
```
- オファーを提示する前に、読者が感じているであろう不安や反論を先に言語化
- 「安心してください」で不安を解消
- 読者が「YES」と言いやすい心理状態を作ってからオファー

### 13.7 具体的な痛みの列挙
**「〜したり」でリズムを作る**
```
でも、結局うまくいかず離婚したり、ATMとして使われるだけの男になったり、めちゃくちゃ苦労している人が多いんですよね。
```
- 具体的な最悪のシナリオを3〜5個列挙
- 「〜したり」で語尾を統一してリズムを作る
- 「ATMとして使われる」など、読者が恐れている具体的な状況を描写

### 13.8 焦点の明確化（企業→視聴者）
**企業のミッション説明→視聴者へのメッセージ**

❌ 避けるべき：
```
僕らKRCのミッション、つまり使命は何なのか？
それは、「『幸福の資本論』を具現化するプログラムを提供する」ということです。
対象領域は、社会資本：恋愛・結婚・夫婦・子育て、人的資本：美容・ファッション…
```

✅ 改善後：
```
そして、最後に、最もあなたに伝えたいことがあります。
僕のミッションは、「愛する一人の女性を幸せにできる真の優しい男を増やす」ことです。
```

- 企業の全体像説明ではなく、視聴者に直接関係するメッセージに絞る
- 「僕ら」「KRC」ではなく「僕」「あなた」を使う
- 抽象的な概念（「幸福の資本論を具現化」）→具体的で感情的な表現（「一人の女性を幸せに」）

### 13.9 冗長な説明の削減
**削減すべき要素**
- ビジョン・バリューの10項目説明
- 対象領域の詳細リスト
- 追伸での同じ内容の繰り返し

**結果**
- ミッション・ビジョン動画：8,500文字→6,800文字（20%削減）
- 個別相談誘導動画：7,200文字→5,500文字（24%削減）
- 視聴者の集中力を維持
- 重要なメッセージに焦点を絞る

### 13.10 自分の体験談の具体化
**抽象論→具体的な結果**
```
逆に、理想のパートナーと出会えれば、毎日が幸せで満たされ、仕事も人生も前向きになります。
僕も今の妻と出会ってから、収入が一気に上がりました。
```
- 一般論で終わらせず、自分の体験を追加
- 数値や結果（「収入が上がった」）を示して説得力を高める
- 読者が「自分もそうなれるかも」と思える
