# インスタグラム運用 実践アクションプラン

## 目標
恋愛コーチとして「集客・コンサル・講座販売」を目的としたInstagram運用

---

## フェーズ1：アカウント準備・立ち上げ（1週間）

### ステップ1-1：プロフィール設計（Day 1-2）

#### ① プロフィール画像
- [ ] 顔出しありの写真を準備
- [ ] 以下の条件を満たす写真を選定：
  - 被写体の顔がしっかり見える
  - 背景や構図がすっきりしている
  - 全体的にクオリティの高い印象を与える
- [ ] 必要に応じてプロに撮影依頼

#### ② 名前の設定
- [ ] 以下のフォーマットで名前を設定：`名前｜コンセプト`
- [ ] 例を参考に具体的なコンセプトを決定：
  - けいじ｜"本命女性"専門 恋愛コーチ
  - けいじ｜唯一無二の男になる恋愛戦略家
  - けいじ｜モテずに愛される恋愛コーチ
- [ ] 自分のコンセプトを決定して名前を設定

#### ③ プロフィール文
- [ ] 5行以内で簡潔にまとめる
- [ ] 以下の要素を含める：
  - 自分が何者か
  - どんな価値を提供するか
  - ターゲット（誰の役に立つか）

#### ④ ハイライト準備（初期版）
- [ ] 「自己紹介」ストーリーを作成：
  - もともとは非モテだったこと
  - どのように変化したか
  - 現在の状況（○○人とのデート実績など）
- [ ] ビフォーアフターのストーリーを3-5個作成
- [ ] ハイライトにまとめて設置

### ステップ1-2：他プラットフォームのアカウント準備（Day 2-3）

- [ ] TikTokアカウントを開設
- [ ] YouTubeショートアカウントを開設（YouTubeチャンネル）
- [ ] 各プラットフォームで同じプロフィール情報を設定
- [ ] 相互リンクを設定

### ステップ1-3：スマホ設定の最適化（Day 3）

#### iPhone設定
- [ ] 設定 > カメラ > ビデオ撮影 > 4K/60fps に設定
- [ ] インカメは使用しない（画質が落ちる）
- [ ] 外カメラでの撮影を徹底

## フェーズ2：AIジャンル認知施策（開始〜2-3ヶ月継続）

### ステップ2-1：同ジャンルアカウントのフォロー（Day 4-5）

- [ ] 恋愛発信で伸びているアカウント（フォロワー1万人以上）を10〜20アカウント特定
- [ ] リストアップしたアカウントをフォロー
- [ ] 各アカウントの投稿スタイルをメモ

### ステップ2-2：日々のジャンル認知施策（毎日実施）

#### 必須アクション（毎日）
- [ ] フォローした同ジャンルアカウントの投稿に「いいね」（1日10-20件）
- [ ] フォローした同ジャンルアカウントの投稿を「保存」（1日5-10件）
- [ ] 同ジャンル発信者にDMを送る（1日3-5件）
  - 返信がなくてもOK
  - AIにジャンル認知させることが目的

#### インサイト確認
- [ ] 週1回、インサイトで「発見」という表示が出ているか確認
- [ ] 「発見」が表示されるまで上記施策を継続

#### 追加施策
- [ ] 同ジャンルのリールが流れてきたら「興味あり」をタップ
- [ ] 同ジャンル以外のリールが流れてきたら「興味なし」をタップ

#### NG行動（絶対に避ける）
- [ ] 同ジャンル以外のアカウントと絡まない
- [ ] いいねは同業のみに限定
- [ ] 他ジャンルの投稿に反応しない

## フェーズ3：コンテンツ制作・投稿開始（Day 7〜）

### ステップ3-1：コンテンツリサーチ（継続的に実施）

#### リサーチ手順
- [ ] TikTok、Instagram、YouTubeで以下のキーワードで検索：
  - 恋愛
  - 非モテ
  - 彼女欲しい
  - モテテクニック
  - 恋愛コーチ
- [ ] 30万回以上再生されている動画をリストアップ（10-20本）
- [ ] リサーチノートを作成：
  - 動画ネタ・企画・動画構成
  - 動画の秒数
  - 動画のカット秒数
  - 動画の撮り方
- [ ] 再生数の多い順に優先順位をつける

### ステップ3-2：伸びる動画構成の習得

#### 構成パターン1：比較・言い換え系
- [ ] 「NG例 vs OK例」形式の動画構成を研究
- [ ] 「〇〇な男の言い方」系の動画を分析
- [ ] テンプレートを3パターン作成

#### 構成パターン2：カメラに語りかける形式
- [ ] 参考アカウント（土橋さん、やまもとりゅうけんさん）の動画を10本以上視聴
- [ ] 以下の要素を取り入れる：
  - フォロワーからの質問に答える形式
  - 逆張りフック（「〇〇しないでください。なぜなら〜」）
  - 自分の経験談を交える
- [ ] トークスクリプトのテンプレート作成

#### 構成パターン3：表形式×ショート動画
- [ ] 5秒程度の短い動画フォーマットを研究
- [ ] Canvaなどで表形式のテンプレート作成
- [ ] 情報を簡潔にまとめる練習

### ステップ3-3：動画制作（週2-3本）

#### 制作前チェックリスト
- [ ] リサーチで見つけたバズっている動画ネタを参考にする
- [ ] 自己流は避け、伸びている型を使う
- [ ] 台本を作成（丸パクリはNG、型とノウハウを参考）

#### 撮影
- [ ] 外カメラで撮影
- [ ] 4K/60fps設定を確認
- [ ] 照明・音声の確認

#### 編集
- [ ] CapCutやInShotなどの編集アプリを使用
- [ ] テキスト、効果音、BGMを追加
- [ ] 動画の長さを調整（15-30秒推奨）

#### アップロード前
- [ ] リール投稿画面で「その他のオプション」をタップ
- [ ] 「最高画質でアップロード」を選択

### ステップ3-4：キャプション作成

- [ ] 1行目に必ず以下を記載：
  ```
  @ユーザーネーム ←他の投稿はこちらから
  ```
- [ ] 2行目以降に投稿の説明を簡潔に記載
- [ ] 最大5行まで

### ステップ3-5：ハッシュタグ設定

- [ ] 5〜10個のハッシュタグを付ける
- [ ] 以下のハッシュタグから選択：
  ```
  #恋愛相談 #恋愛コーチ #彼女の作り方 #本命女性
  #非モテ脱出 #彼女が欲しい #自分に自信を持ちたい
  #婚活男子 #恋愛がうまくいかない #LINE苦手
  #デート成功率アップ #一途な男
  #好きな人に振り向いてもらいたい #ナンパじゃなくて恋愛
  #恋愛心理学 #男の恋愛戦略 #愛される男になる
  #恋愛講座 #恋愛で人生を変える
  ```
- [ ] 大きいタグ数（30万件以上）と小さいタグ数（10万件以下）を混ぜる

### ステップ3-6：3プラットフォームへの同時投稿

- [ ] Instagram リールに投稿
- [ ] TikTokに投稿（同じ動画）
- [ ] YouTubeショートに投稿（同じ動画）
- [ ] 投稿時間を記録

## フェーズ4：投稿ルーティンの確立（継続）

### 週間スケジュール

#### 毎日のタスク
- [ ] ストーリー投稿（できれば毎日、最低でも2-3日に1回）

- [ ] 同ジャンルアカウントへのエンゲージメント（いいね・保存・DM）

#### 週2-3回のタスク
- [ ] リール投稿（火・木・土など決めた曜日）
- [ ] インサイト分析（どの投稿が伸びたか確認）

#### 週1回のタスク
- [ ] 週次レビュー：
  - フォロワー増加数
  - 再生回数が多かった投稿
  - エンゲージメント率
  - 「発見」に載っているか確認
- [ ] ハイライト更新（質問コーナーなど）

#### 月1回のタスク
- [ ] 月次レビュー：
  - 月間のフォロワー増加数
  - 最もバズった投稿の分析

## 重要な投稿頻度ルール

### 絶対に守るべき頻度
- ✅ **ストーリー**：毎日（最低2-3日に1回）
- ✅ **リール**：週2-3回（3日以上空けない）
- ✅ **フィード投稿（画像）**：不定期でOK（ほぼ不要）

### 注意事項
- ⚠️ アカウント初期は3日以上空くとエンゲージメントが落ちる
- ⚠️ 投稿頻度の安定性が重要
- ⚠️ リール主体（リール9：フィード1の比率）

---

## 成功の3大原則

### 1. 日々のジャンル認知施策
- 同ジャンルアカウントへのいいね・保存・DM
- これを怠ると伸びない

### 2. 投稿頻度の厳守
- リール：週2-3回
- ストーリー：毎日〜2-3日に1回
- 3日以上空けない

### 3. 伸びているコンテンツから参考にする
- 自己流は避ける
- 30万回以上再生の動画を研究
- 型とノウハウを学ぶ（丸パクリはNG）

## トラブルシューティング

### 伸びない場合の確認項目
1. [ ] AIジャンル認知施策を毎日行っているか？
2. [ ] インサイトで「発見」が表示されているか？
3. [ ] 自己流の投稿になっていないか？
4. [ ] リサーチした伸びている動画を参考にしているか？
5. [ ] 投稿頻度を守っているか？
6. [ ] 同ジャンル以外のアカウントと絡んでいないか？
7. [ ] 動画の画質設定は最高になっているか？
8. [ ] ハッシュタグを適切に使っているか？

### よくある失敗パターン
- ❌ 自己流の発信（伸びている型を学ばない）
- ❌ ジャンル認知施策をサボる
- ❌ 投稿頻度が不安定
- ❌ 同ジャンル以外のアカウントと絡む
- ❌ 画質設定を最適化していない
- ❌ フィード投稿（画像）ばかり投稿する

---

## まとめ：成功への最短ルート

1. **準備を完璧に**：プロフィール・スマホ設定・他プラットフォーム
2. **ジャンル認知施策を毎日**：これが最重要
3. **伸びている動画を研究**：自己流は捨てる
4. **投稿頻度を守る**：リール週2-3回、ストーリー毎日
5. **3プラットフォームで同時発信**：特にTikTokは必須
6. **インサイトを確認**：「発見」に載るまで改善
7. **継続する**：最低3ヶ月は続ける

この順番で実践すれば、確実に成果が出ます。頑張りましょう！
