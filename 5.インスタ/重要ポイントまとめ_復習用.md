# インスタグラム運用 重要ポイントまとめ（復習用）

---

## 🎯 最重要：成功の3大原則

### 1️⃣ 日々のジャンル認知施策 ⭐️⭐️⭐️
**これを怠ると絶対に伸びない**
- 同ジャンルアカウント（1万人以上）へ毎日エンゲージメント
- いいね・保存・DM（返信不要）
- インサイトで「発見」が表示されるまで継続
- **AIに認知されないと一生伸びない**

### 2️⃣ 投稿頻度の厳守 ⭐️⭐️
**3日以上空けると終わり**
- リール：週2-3回（必須）
- ストーリー：毎日〜2-3日に1回
- 3日以上空くとエンゲージメント激減

### 3️⃣ 伸びているコンテンツから学ぶ ⭐️⭐️
**自己流は絶対NG**
- 30万回以上再生の動画を研究
- 型とノウハウをパクる（丸パクリはNG）
- 自分の良いと思う動画≠人から見た良い動画

## 📊 2024年Instagram攻略法

### リールが全て
- **リール9：フィード1** の比率で投稿
- 2024年から爆伸びしているアカウントの9割以上がリール主体
- 画像投稿（フィード）は伸びない時代

### 3プラットフォーム同時投稿
1. **Instagram リール**
2. **TikTok**（特に重要・拡散力が強い）
3. **YouTubeショート**

→ 同じ動画を3つに投稿できるので効率的
→ TikTokでバズるとInstagramにも流入

## 🔧 プロフィール設計の鉄則

### ① プロフィール画像
✅ 顔出しあり（被写体が近め）
✅ 背景すっきり
✅ 高品質な写真
❌ インカメNG（画質が落ちる）

### ② 名前
**フォーマット**：`名前｜コンセプト`

**Good例**：
- けいじ｜"本命女性"専門 恋愛コーチ
- けいじ｜唯一無二の男になる恋愛戦略家

**NG例**：
- けいじ｜恋愛コーチ（抽象的すぎる）

### ③ ハイライト
**必須コンテンツ**：
- 自己紹介（過去の黒歴史）
- ビフォーアフター
- 質問コーナー（500-1000人超えたら）

### ④ プロフィール文
- **5行以内**で簡潔に
- 自分が何者か、何を提供するか明確に

## 🤖 AIジャンル認知の仕組み

### なぜ必要？
Instagram AIに「このアカウントは恋愛ジャンル」と認知されないと、**どれだけ投稿しても伸びない**

### 認知されたサイン
インサイトで「**発見**」という項目が表示される
→ これが出たらアカウントが爆発的に伸び始める

### 認知施策（毎日やること）
1. 同ジャンル1万人以上のアカウントを10-20フォロー
2. それらのアカウントに毎日：
   - いいね（10-20件）
   - 保存（5-10件）
   - DM（3-5件）※返信不要
3. 同ジャンルのリールに「興味あり」
4. 他ジャンルのリールに「興味なし」

### 絶対NG
❌ 同ジャンル以外のアカウントと絡む
❌ 他ジャンルの投稿にいいね
→ AIが混乱して認知されなくなる

## 🎬 リール制作の極意

### 画質設定（超重要）
**iPhone設定**：
1. 設定 > カメラ > ビデオ撮影 > **4K/60fps**
2. 外カメラで撮影（インカメNG）

**投稿時**：
1. リール投稿画面
2. 「その他のオプション」
3. 「**最高画質でアップロード**」を選択

### キャプションの書き方
```
@ユーザーネーム ←他の投稿はこちらから

【投稿の説明を簡潔に】
（最大5行まで）
```

**理由**：1行目にユーザーネームを入れると、プロフィール訪問率が上がる→AIから高評価

### ハッシュタグ
- **5-10個**を付ける
- 大きいタグ（30万件以上）+ 小さいタグ（10万件以下）を混ぜる

**推奨タグ**：
```
#恋愛相談 #恋愛コーチ #彼女の作り方 #本命女性
#非モテ脱出 #彼女が欲しい #自分に自信を持ちたい
#婚活男子 #恋愛がうまくいかない #LINE苦手
#デート成功率アップ #一途な男
#好きな人に振り向いてもらいたい #ナンパじゃなくて恋愛
#恋愛心理学 #男の恋愛戦略 #愛される男になる
#恋愛講座 #恋愛で人生を変える
```

## 💡 バズる動画構成（3パターン）

### パターン1：比較・言い換え系
**最もバズりやすい**
- 「NG例 vs OK例」
- 「〇〇な男の言い方」
- 「たった一言で意識される言い換え」

**特徴**：
- 短時間で情報を伝える
- 視覚的にわかりやすい
- 何度も見返される

### パターン2：カメラに語りかける
**信頼感が最も伝わる**
- フォロワーからの質問に答える形式
- 逆張りフック（「〇〇しないでください。なぜなら〜」）
- 自分の経験談を交える

**参考アカウント**：
- 土橋さん
- やまもとりゅうけんさん

### パターン3：表形式×ショート動画
**最近激バズ中**
- 5秒程度の短い動画
- 情報を表にまとめる
- 何度も見返されやすい→アルゴリズム評価高

**例**：
- お風呂のNGの声掛け
- 脳が育つ声かけ
- 転職で後悔しない選び方

## 🔍 コンテンツリサーチの方法

### リサーチ場所
- TikTok
- Instagram
- YouTube

### リサーチキーワード
```
恋愛、非モテ、彼女欲しい、モテテクニック、恋愛コーチ
```

### 基準
✅ **30万回以上**再生されている動画をピックアップ
✅ 再生数が多いほど需要が高い
✅ 迷ったら再生数の多い方を参考にする

### リサーチノートに記録
1. 動画ネタ・企画・動画構成 **←最重要**
2. 動画の秒数
3. 動画のカット秒数（1カットの秒数）
4. 動画の撮り方

### 参考にする3つの基準

| レベル | 方法 | 評価 |
|--------|------|------|
| 三流 | 丸パクリ | ❌ NG |
| **二流** | **型やノウハウをパクる** | ⭐️ **推奨** |
| 一流 | 感覚をパクる | ⭐️⭐️ 理想 |

→ まずは**二流**を目指す
→ 型を学び続けると自然に感覚が身につく

## ⚠️ よくある失敗パターン

### ❌ 絶対にやってはいけないこと

1. **自己流の発信**
   - 自分が良いと思う動画を投稿
   - 伸びている型を学ばない
   → 結果：300回再生で停滞

2. **ジャンル認知施策をサボる**
   - 同ジャンルへのエンゲージメントなし
   → 結果：AIに認知されず永遠に伸びない

3. **投稿頻度が不安定**
   - リールを週1回以下しか投稿しない
   - 3日以上空ける
   → 結果：エンゲージメント激減

4. **同ジャンル以外と絡む**
   - 他ジャンルのアカウントにいいね
   - 様々なジャンルにコメント
   → 結果：AIが混乱してジャンル認知されない

5. **フィード投稿（画像）ばかり**
   - リールを投稿しない
   - 綺麗な写真を投稿
   → 結果：2024年のアルゴリズムでは伸びない

6. **画質設定の最適化をしない**
   - インカメで撮影
   - 4K設定にしていない
   → 結果：AIから低評価

- 競合分析は成功の鍵

## 🎓 マインドセット

### 伸びない人の共通点
❌ 自己流の発信
❌ ジャンル認知施策をしていない
❌ 投稿頻度が守れない

### 伸びる人の共通点
✅ 素直に学ぶ（型を真似る）
✅ 毎日のエンゲージメント施策
✅ 投稿頻度を守る
✅ 継続する（最低3ヶ月）

### 覚えておくべきこと
> 「自分の良いと思う動画」と「人から見た良い動画」は大きく違う

> 「伸びている動画 = それだけ需要とニーズが高い」

> 「AIに認知されないことには、始まらない」

## 🔥 モチベーション維持のために

### 成功事例を思い出す

### 諦めない

- AIに認知されるまで（2-3ヶ月）は地道な作業
- 「発見」に載り始めたら爆発的に伸びる
